#### schema_map_production_datascience_casestudy

A NLP Study Project in Two Parts

# Extended Project
- Scenario: Old MySQL HRM database records need to be migrated to a new MongoDB document-database. Task: Make a tool to analyze the two schemas and suggest a schema-map.
- Part 1: Make a Python LLM solution: Working code, docs, sample output.
- Part 2: Make a Rust GOFAI-NLP solution: Working code, docs, sample output.
- In documentation, discuss production data scince and software factors and design choices, compare and contrast, and discuss the relationship between, the two above approaches.
